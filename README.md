# FitHub-Admin
FitHub Admin este o aplicație pentru administrarea unei săli de fitness, oferind funcționalități de gestionare a clienților, abonamentelor, antrenorilor și claselor. Sistemul folosește un REST API în .NET și o bază de date SQLite pentru persistență, fiind containerizat cu Docker pentru rulare ușoară.

Sistemul se bazează pe o arhitectură REST API implementată în .NET, folosind o bază de date SQLite pentru persistența datelor. Prin intermediul interfeței sale, operatorii sălii pot realiza rapid operații CRUD asupra entităților principale, facilitând evidența abonamentelor active, programărilor la clase și listei de antrenori disponibili.

FitHub Admin este proiectat astfel încât să fie ușor de extins și de integrat, fiind containerizat cu Docker pentru o distribuire rapidă și o rulare uniformă în orice mediu. Principalul obiectiv al aplicației este oferirea unui sistem stabil, scalabil și intuitiv, care să sprijine activitatea administrativă a sălilor de fitness de orice dimensiune.
